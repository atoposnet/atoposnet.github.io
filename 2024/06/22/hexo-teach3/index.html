在 Github 中添加 SSH 密钥

在本地生成 ssh 密钥

（1）首先在 home 目录创建 .ssh 目录

（2） 使用 ssh-keygen 指令生成 ssh 密钥

（2）生成的密钥保存在 ~/.ssh/id_rsa.pub文件中，我们可以使用 cat id_rsa.pub 命令查看生成的 ssh 密钥

把本地生成的 ssh 密钥拷贝到 github 中

（1）首先进入 github 官网并登录账号，点击右侧的用户头像图标

（2）在下拉的选项中点击 “Settings” 进入设置页面

（3）点击 “Setting” 页面左边的 “SSH and GPG keys” 选项，并通过 “New SSH key” 添加 SSH key

（4）拷贝本机上的 SSH 密钥到图中红框的位置，并点击 “Add SSH key” 完成 SSH key 的添加

（5）SSH key 添加成功后可以在 “SSH and GPG keys” 页面看到添加的 “SSH key” 信息，如图所示

验证 SSH key

在 github 中查看仓库的 clone 地址

（1）回到 github 主页，点击右侧的用户头像图标，在下拉的选项中选择 “Your repositories” 进入你的仓库列表

（2）在仓库的列表中点击我们创建的托管仓库

（3）在我们的仓库中点击 Code 查看仓库的 Clone 地址，注意这里我们使用 SSH 方式的 Clone 链接
