创建 Github 账号

我们基于 hexo 搭建的网站最终要通过 github 进行托管，所以首先需要注册一个 github 账号，具体方法参考网上教程。

创建 Github 托管仓库

创建托管仓库和普通仓库的唯一区别就是仓库名称，托管仓库的名称必须是 “github用户名.github.io” 格式。

（1）首先进入 github 官网并登录账号，点击右侧的用户头像图标

（2）在下拉的选项中点击 “Your repositories” 进入你的仓库列表

（3）对于新注册的账号，仓库列表为空，此时我们点击 ”New“ 创建仓库

（4）由于我们要创建的是托管仓库，因此仓库的名称需要按照图中的格式命名

（5）仓库类型我们选择 “Public”, 并选中 “Add a README file”, 然后点击 “Create repository” 完成托管仓库的创建

（6）托管仓库创建完成后如下图所示，我们点击 “Setting” 查看托管仓库的信息

（7）点击 “Setting” 页面左边的 “Pages” 选项，可以看到我们托管仓库的域名

（8）我们在浏览器的地址栏输入托管仓库的域名，可以看到该页面显示了我们仓库中的 README.md 文件

总结

至此，我们已经完成 github 托管仓库的创建，接下来我们需要实现在本地上传和下载仓库中的文件。
